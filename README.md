# CPAS 初級培訓｜隱藏背景互動模擬系統

這是一個可直接放在 GitHub Pages 的靜態網頁版本。

## 檔案說明
- `index.html`：主網頁
- `style.css`：版面樣式
- `data.js`：題組、個案回答、CPAS 對照資料
- `app.js`：互動邏輯、資料格、CPAS 揭示、報告骨架產生器

## 使用方式
1. 開啟 `index.html`。
2. 第一階段「提問蒐證」不顯示 CPAS，學員透過選題或自訂提問取得個案回答。
3. 將可用資料加入資料格。
4. 第二階段揭示 CPAS，回頭檢查哪些晤談資料能支持 CPAS 判讀。
5. 第三階段產出報告骨架。

## 教學提醒
這個版本的重點不是讓學員直接得到答案，而是練習：
- 問出背景與現況
- 判斷哪些資料能放進報告
- 刪除不必要資料
- 從晤談資料收斂職涯問題
- 用 CPAS 分數回頭對照行為證據
- 讓建議對應職涯問題與 CPAS 指標
