# CPAS 初級培訓｜隱藏背景互動模擬系統 v1.3.5

此版本針對「第三階段：報告骨架」進行更新，讓每一個報告欄位都有提示文字、填寫提示、可展開的額外說明，以及學員可填寫的文字框。

## 使用方式

1. 解壓縮 ZIP。
2. 開啟 `index.html`。
3. 先進行提問蒐證。
4. 整理資料格。
5. 揭示 CPAS 對照。
6. 到「④ 報告骨架」完成報告練習。
7. 可使用「複製報告骨架」或「下載PDF檔」。

## GitHub Pages

將所有檔案上傳至 GitHub repository 根目錄後，在 Settings → Pages 啟用 Deploy from a branch，選擇 main / root。


## v1.3.5 更新重點

- 驚嘆號 `!`：展開填寫提示。
- 問號 `?`：展開額外提示說明。
- 報告骨架更接近課堂操作版，預設畫面更乾淨。


## v1.3.5 注意事項

本版恢復教學流程鎖定：第三部分與第四部分預設隱藏，必須先按下「揭示 CPAS 對照資料」才會開放。


## v1.3.5 補充
報告骨架頁一開始會顯示所有項目與空白撰寫框。資料格整理後會自動帶入部分內容；揭示 CPAS 後才會帶入 CPAS 相關欄位。手動修改內容會保留。
